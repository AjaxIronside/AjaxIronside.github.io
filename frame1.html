System7.css

An open-source, retro-style web theme using HTML, CSS and JavaScript, by Al Such.

Hello!

Thanks for taking a look at this, my first-ever CSS theme.
I'm definitely still learning the ropes, but I hope that this gives the retro enthusiasts in the community a smile, and that this shows just how much I have to offer.

This theme uses iFrames to display content in this "Main Window" section of the page, similar to the Macintosh System 7 operating system. To experiment with the changing frames, see the example dropdown menu above.

Additionally, to download the theme, or to see some more about me, check out the links in the "Finder" bar!

Thanks again for checking out my first ever theme.

-Al
